# Blog App

> This project is a Food Recipe application built with Ruby and Ruby on Rails

## Built With

- Ruby
- Ruby on Rails
- CSS

## Getting Started

```
git clone https://github.com/aliabbani/Recipe-group-

cd Recipe-group-

rails db:create
rails db:migrate
rails db:seed

rails server
```

## Tests

```
git clone https://github.com/aliabbani/Recipe-group-
cd Recipe-group-
bundle install
gem install rspec
rspec
```
